## Getting Started

In the file called:
```
.env.local[duplicate]
```

remove [duplicate] so file is called ```.env.local```

Install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result
